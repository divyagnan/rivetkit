Metadata provides information about the currently running actor.

## Region

Region can accessed from `this._region`.

<Warning>`this._region` is only supported on Rivet at the moment.</Warning>

## Tags

Tags can be accessed from `this._tags`.

For example:

<CodeGroup>

```typescript chat_room.ts
import { Actor, type Rpc } from "actor-core";

export default class ChatRoom extends Actor {
  // Method used to get the channel ID. Returns `random` for this example.
  getChannelId(rpc: Rpc<ChatRoom>): string {
    return this._tags['channel'];
  }
}
```

```typescript client.ts
// Create an actor with `channel` set to `random`
const actor = client.get<ChatRoom>({
  name: 'chat_room',
  channel: 'random'
});
```

</CodeGroup>
